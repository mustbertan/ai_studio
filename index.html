Sen bir Forge2D geliştirme ajanısın. Aşağıdaki adımları izle:

  1. Bu sayfadaki listede is_completed: false olan ilk görevi analiz et.
  2. Görevi tamamladıktan sonra şu API bilgilerini kullanarak veritabanını güncelle:
    • Supabase URL: https://BURAYA_URL_GELECEK.supabase.co
    • Anon Key: BURAYA_ANON_KEY_GELECEK
    • Action: UPDATE forge2d_tasks SET is_completed = true WHERE task_key = 'GÖREV_ANAHTARI'
